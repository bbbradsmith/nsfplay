NSFPlay

NSFPlay is an open source NSF player for Windows and plugin for Winamp (NSFPlug).
It's a program for listening to NES music.

  • Stable release: NSFPlay 2.6 2023-07-03
  • Unstable preview: Github Actions Artifact
  • Source code: GitHub Repository

NSFPlay was originally created by Brezza, who continued development until 2004.
This fork was continued by rainwarrior in 2012 to add support for modern Windows,
improve accuracy, and otherwise continue making improvements.
